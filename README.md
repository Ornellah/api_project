# api_project

Projet bac à sable ayant pour but de pouvoir appliquer des techniques et connaissances.   
    
Ce projet est constitué d'une API avec un front mais qui peut aussi fonctionner sans front-end.   
Deux modèles "scolaires" (utilisation d'algorithmes basiques non pertinent sur les datasets "titanic" et "iris") sont implémentés.     
Le but actuel n'étant pas de créer un modèle mais de savoir implémenter des modèles dans une API.    
Deux tests unitaires basiques ont été implémentés pour vérifier la sortie des fonctions de prédiction.      
La CI/CD a été implémentée mais sera potentiellement revue.     
Le projet a été déployé sur vercel avec succès dans son ancienne version mais était trop volumineux (dû aux données et modèles et à l'utilisation de la version gratuite).
